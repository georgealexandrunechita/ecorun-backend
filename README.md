# EcoRun

EcoRun es una aplicación web para gestionar y visualizar rutas de running, con enfoque en sostenibilidad y hábitos saludables.

## Tecnologías

- Node.js y Express (backend)
- MySQL (base de datos)
- React / Vite (frontend) *(ajusta si usas otra cosa)*
- Git y GitHub para control de versiones
